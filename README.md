## Project Overview
This project classifiesIris flowers into 3 species - Setosa, Versicolor, Virginica using Machine Learning for CodeAlpha Internship Task 1.

## Tech Stack
-Python
-Pandas,Numpy
-Scikit-Learn
-Matplotlib,Seaborn

##Model & Accuracy
-**Algorithm**: Random Forest Classifier
-**Accuracy**: 100% on test data
_**Dataset**: Iris Dataset from scikit-learn

##Files
1.'CodeAlpha-Iris Flower Classification.ipynb' - Main Jupytor Notebook
2.'Iris.csv' - Dataset file
